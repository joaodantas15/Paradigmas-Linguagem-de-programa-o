Matemática & Lógica

Inclusão de divisão (/), exponenciação (^), operadores ou / xor e relações <>, <, >, >=.

Estruturas de Controle

Novos comandos para (loop contado), repita (loop fixo), escolha (switch case) e suporte a múltiplos senaose.

Sintaxe & I/O

Ponto e vírgula (;) como finalizador, atribuição paralela e comando exiba adaptado para números e textos.

Demonstração das Tarefas Realizadas

Atribuição Paralela & Escolha

// Inversão de valores e Switch
a, b := 10, 20;
a, b := b, a;

escolha a
  10 : exiba "Dez";
  20 : exiba "Vinte";
  _  : exiba "Outro";
;

Múltiplos Senaose

se nota >= 6 entao
  exiba "Aprovado";
senaose nota >= 3 entao
  exiba "Recuperação";
senao
  exiba "Reprovado";
;

Novos Loops (Para/Repita)

// Contagem de 1 a 5

para i de 1 ate 5 faca
  escreva i;

// Repetição fixa

repita 10 vezes
  exiba "Executando...";

Lógica de Implementação (TS)

Uso de Math.pow e Math.floor para novas operações.

case '^': return Math.pow(left, right);
case '/': return Math.floor(left / right);
